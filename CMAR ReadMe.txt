The CMAR program includes 4 Python files, "Part5_FP_Tree.py", 
"Part5_CR_Tree.py", "Part5_Classifier.py" and "Part5_CMAR_Main.py".

To run the program, you may run the "Part5_CMAR_Main.py". Then 
it will print all the results for the 7 chosen datasets in the console.

In case the program returns an error, please kindly run the program
again.
