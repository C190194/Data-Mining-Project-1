The CMAR program includes 4 Python files, "new_FP_Tree.py", 
"new_CR_Tree.py", "new_Classifier.py" and "new_CMAR_Main.py".

To run the program, you may run the "new_CMAR_Main.py". Then 
it will print all the results for the 7 chosen datasets in the console.

In case the program returns an error, please kindly run the program
again.
